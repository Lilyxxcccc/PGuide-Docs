1. 在教育部出具的[《全国普通高校大学生竞赛分析报告》](https://rank.moocollege.com/)中出现的赛事，包括观察中的赛事；
@[pdf height="600px"](https://cos.cqmu.online/docs/competition2024.pdf)
2. 在重医卓越计划中包含的赛事；
@[pdf height="600px"](https://cos.cqmu.online/docs/cqmu-competitions.pdf)
3. 对个人成长、团队合作有帮助的赛事；
4. 编写成员参加过的赛事。
