## CQMUA是什么？

CQMUA，全称Chongqing Minecraft University Alliance，是由重庆市高校的Minecraft服务器联盟。


## 项目简介

服务器中心的使命是为重庆高校的Minecraft玩家提供一个服务器的状态监控界面。

## 项目技术

- 服务器中心前端：使用Vue.js开发，提供服务器状态监控界面。
