## 复原工程

复原工程是一项宏大的项目，渝高联早在2024年10月就已讨论过联合复原的可能性。模式上参考了福建省闽高联。过去的半年内，管理员一直在进行基础设施的建设和规划，也在积极向其他高校寻求建议与合作。
