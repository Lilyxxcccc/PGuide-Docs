## Onion NetWork
洋葱网络译为 Onion Network，Tor Broswer 是专门为隐私访问设计的一款浏览器，在用户访问国外网站时，不会直接请求国外主机的域名解析地址，而是重新设计路由策略，分为 3 次公共主机节点跳转访问。
