# 项导文档
## 依赖环境
- Node.js v18.20.0+ 
- pnpm 8+ 或 Yarn 2+

- 推荐使用nvm做Node.js环境管理
## 项目安装
- 使用nvm安装npm(以版本18.20.0为例)
    ```shell
    nvm install 18.20.0
    ```
- 全局安装pnpm
    ```shell
    npm install -g pnpm
    ```
- 安装pnpm依赖
    ```shell
    pnpm i
    ```
- 启动项目
    ```shell
    pnpm run:dev
    ```
